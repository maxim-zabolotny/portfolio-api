## Installation

```bash
$ npm install
```

## Run postgres

```bash
# with docker
$ npm run docker-up
```

## Run migration

```bash
$ npm run migration:run
```

## Running the app

```bash
# watch mode
$ npm run start:dev
# production mode
$ npm run start:prod
```
